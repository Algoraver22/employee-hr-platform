# 🧑‍💼 Employee HR Platform

A full-stack HR management system built using the MERN stack. This platform enables HRs and team leads to manage employee records, leave applications, performance metrics, and internal communication seamlessly through a modern, secure, and responsive web interface.

---

## 🌐 Tech Stack

**Frontend:** React.js, Tailwind CSS, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT, Bcrypt.js  
**Other Tools:** React Router DOM, Axios, Context API

---

## 📌 Features

- 🔐 **Secure Authentication & Authorization**
  - Login/Signup functionality using JWT
  - Role-based access (Admin, HR, Employee)

- 👥 **Employee Management**
  - Add, edit, delete employee details
  - View and search employee list
  - Assign departments, roles, and teams

- 📆 **Leave & Attendance**
  - Leave application workflow
  - HR approval/rejection system
  - Attendance record tracking (optional)

- 📊 **Performance & Reports**
  - Submit and review employee performance
  - View analytics dashboards (planned)

- 📨 **Internal Communication**
  - HR notices & announcements module (planned)
  - Optional team chat integration (future scope)

---

## 🚀 Getting Started

### 1. Clone the Repository

git clone https://github.com/Algoraver22/employee-hr-platform.git
cd employee-hr-platform

---

### 2. Set Up Backend

cd server
npm install

---

### 3. Set Up Frontend

cd ../client
npm install
npm start
